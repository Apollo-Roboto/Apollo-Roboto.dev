# Apollo-Roboto.dev-source

This is the source code of my personal website at [Apollo-Roboto.dev](https://www.apollo-roboto.dev/).
Powered with Hugo and github pages.

## Run locally

run the development server with:
```bash
hugo server -D
```
Note that the generated static pages in the public folder won't work as intended as the link to resources will be different.

## Github workflow

A workflow has been setup to automatically upload updates to the [Apollo-Roboto.dev repository](https://github.com/Apollo-Roboto/Apollo-Roboto.dev)'s github pages
